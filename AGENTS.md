# تعليمات دائمة للمساعد

## أسلوب التفاعل

- **لا تطرح أسئلة توضيحية ولا تعرض خيارات بديلة قبل التنفيذ.** نفّذ المطلوب مباشرة باعتمادك قراراً معقولاً عند الغموض، ثم اشرح ما فعلته باختصار في نهاية الرد.
- **لا تقدّم اقتراحات إضافية** خارج نطاق ما طُلب صراحة. لا تُذيّل الردود بـ "هل تريد أيضاً..." أو "يمكن تحسين..." إلا إذا طلب المستخدم رأياً أو خيارات.
- **اللغة**: ردود عربية موجزة، إلا عند طلب تغيير اللغة صراحة.
- **التركيز**: نفّذ فقط ما طُلب — لا تعدّل ملفات أو سلوكاً خارج نطاق الطلب.

## قواعد عامة للمشروع

- المصطلح **"إعادة التنظيم"** غير مستخدم في النظام. كل ما يخص هذه المرحلة يُسمّى **"مرحلة مسارات التقييم"**.
- بيئة Python في `.venv` (Python 3.14). التشغيل عبر `run.bat` أو `run.py`.
- ملف بيانات SQLite الافتراضي: `exercises.db` في جذر المشروع.
- عند تعديل `style.css` رفع `?v=…` في `app/templates/base.html` لكسر كاش المتصفح.
